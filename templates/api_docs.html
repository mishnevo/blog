MyBlog API v1.0 — полный доступ к постам через HTTP

Аутентификация

Все запросы к API требуют передачи API-ключа одним из способов:

X-API-Key: {{ api_key }}

Заголовок запроса (рекомендуется)

?api_key={{ api_key }}

Примеры запросов

GET — получить список постов (страница 2):

curl -H "X-API-Key: {{ api_key }}" \
  "http://127.0.0.1:5000/api/posts?page=2&per_page=3&sort=asc"

POST — создать пост:

curl -X POST \
  -H "X-API-Key: {{ api_key }}" \
  -H "Content-Type: application/json" \
  -d '{"title": "Привет!", "content": "Мой первый пост через API."}' \
  http://127.0.0.1:5000/api/posts

DELETE — удалить пост #3:

curl -X DELETE \
  -H "X-API-Key: {{ api_key }}" \
  http://127.0.0.1:5000/api/posts/3
